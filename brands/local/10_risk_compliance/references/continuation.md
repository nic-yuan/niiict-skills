  → 建立促销价格合规审核流程

价格标示合规要点：
  ❌ "原价 99，现价 19"（无法证明原价真实存在）

```
### T10-02 gap: 市场监管低价举报合规应对 SOP

```
触发场景：市场监管部门收到举报，称平台某商品售价低于成本价

背景说明：
  《价格法》第14条：经营者不得以低于成本的价格倾销商品
  但"限时促销"和"拉新补贴"有豁免条件，需准备充分说明材料

处理流程（6步法）：

Step 1：自查确认（2小时内）
  → 核查被举报商品的成本结构：进货价 + 仓储 + 配送 + 包装
  → 确认售价是否真的低于成本（毛利是否为负）
  → 如售价 ≥ 成本价：属于正常促销，无需担心
  → 如售价 < 成本价：进入 Step 2

Step 2：区分促销类型（4小时内）
  类型A：限时秒杀（如：每天10点前10名5折）
    → 合规：有明确时间段和数量限制，不构成持续性低价倾销
    → 需准备：促销活动规则、限购数量、持续时间

  类型B：新用户专享价
    → 合规：针对新用户的优惠，不影响现有用户/供应商利益
    → 需准备：新用户定义、补贴金额来源

  类型C：平台补贴拉新（全场低于成本价）
    → 风险：可能被认定为倾销，需提供补贴来源和营销目的说明
    → 建议：尽快调整策略，降低补贴力度至成本价以上

Step 3：准备说明材料（24小时内）
  必须包含：
    ① 促销活动批准文件（内部审批流程）
    ② 促销活动规则（含时间/数量限制）
    ③ 成本核算表（证明不是持续性低于成本）
    ④ 供应商知情同意书（如有供应商让利）
    ⑤ 以往同类型促销活动的销售记录

Step 4：配合调查（如已上门）
  → 由法务或合规负责人接待，不让普通运营人员直接应对
  → 提供材料时保留副本，记录执法人员的姓名/工号/要求
  → 如有疑问：有权要求查看执法人员证件和调查令

Step 5：申诉与听证（如收到处罚告知）
  → 处罚告知书送达后7天内可申请听证
  → 申诉重点：促销活动具有正当商业目的，非持续性倾销
  → 参考案例：2022年某电商平台"限时特价"申诉成功

Step 6：规则学习与整改
  → 组织全员学习《价格法》和《反不正当竞争法》
  → 建立促销活动合规审核流程（法务+财务双重审核）
  → 今后促销价格不得低于成本价，如需低于成本价需CEO特批

红线预警：
  ❌ 不得虚构原价（先提价再打折）
  ❌ 不得无数量限制的持续性低价倾销
  ❌ 不得强制供应商承担降价损失（否则供应商可举报）
```

### T10-05 gap: 团长劳动仲裁应对 SOP

```
触发场景：团长向劳动仲裁委员会申请劳动仲裁，要求确认劳动关系并补缴社保

背景说明：
  劳动仲裁时效：离职后1年内可申请
  仲裁委认定劳动关系，主要看三要素：人格从属性、经济从属性、组织从属性
  一旦认定劳动关系成立，平台需补缴社保（约占工资30%）+ 可能的双倍工资赔偿

处理流程（分类应对）：

情形A：合同已明确"合作关系"，团长未提供充分反驳证据
  → 评估：胜诉概率高（>70%）
  → 处理：
    1. 法务准备答辩状（重点：合同条款、管理权限、结算方式）
    2. 收集证据（团长考勤/培训记录/管理沟通记录）
    3. 申请调解（劳动仲裁委通常会先调解）
    4. 如调解不成：正常应诉，不必主动妥协

  关键证据准备：
    → 《合作协议》（明确非劳动关系）
    → 团长自行报税记录（团长自行承担税务）
    → 团长无固定考勤打卡记录
    → 团长同时做其他平台的证据（如有）

情形B：团长能证明平台对其存在实际管理（人格从属性强）
  → 评估：败诉风险较高（>50%）
  → 处理：
    1. 法务评估和解成本 vs 败诉成本
    2. 如和解成本 < 败诉成本：主动联系团长协商和解
    3. 和解方案：适当补偿，换取撤诉+保密协议
    4. 如和解不成：应诉，同时做好最坏打算（补缴社保+赔偿）

  常见败诉原因：
    → 团长每天需在群里发固定数量商品（平台管控工作时间）
    → 团长接受平台培训/考核/罚款（平台管控工作方式）
    → 团长收入完全来自平台，无其他收入来源（经济从属性）
    → 团长使用平台提供的专属工具/二维码（组织从属性）

情形C：团长主张"事实劳动关系"（虽签了合作协议）
  → 评估：根据实际情况综合判定
  → 处理：
    1. 分析三要素的满足程度
    2. 如三要素均满足：建议和解（败诉概率 > 80%）
    3. 如仅满足1-2个：可据理力争，但需充分准备

和解谈判话术（法务/HR出面）：
  "关于您的仲裁申请，我们高度重视。
  经过内部审查，我们认为双方是合作关系。
  但考虑到您过去的贡献，我们愿意给予一定补偿。
  具体方案：[补偿金额] + [开具离职证明]
  条件是您撤销仲裁申请，并签署保密协议。"

预防机制（长期）：
  → 合作协议经劳动法律师审核（明确非劳动关系条款）
  → 不对团长进行过度管控（禁止固定排班/强制在线时长）
  → 鼓励团长多平台经营（降低经济从属性）
  → 定期进行团长满意度调研（收集改进建议，降低对立情绪）
  → 咨询劳动法律师，了解最新劳动仲裁判例动态

法律成本估算：
  补缴社保：约为工资总额的30%（如月均工资¥8000，18个月≈¥43,200）
  双倍工资差额：如认定为劳务关系而非劳动关系，约¥72,000
  经济补偿金：约¥12,000（按1个月工资）
  潜在总成本：单个团长约¥127,000
  建议：对高风险团长（月收入>¥5000且全职）提前进行合规评估
